Testing Printer

To test the printer, click Test in the Configurator Printers configuration. The Configurator Printers configuration option is also used to ensure that the device is switched on, properly connected and configured, and ready to print your documents.

 

Check printer

Verifies that the printing system is installed on the printer.

Test device and backend

Verifies that the device is switched on, connected, and can send a response through the backend port.

Print test page

Verifies that pages are printed correctly by printing a test page.

Default test page

Used the build-in test page to verify that pages are printed correctly.

Custom test page

Prints a page you have defined using a specified file name:

  • To select a custom page to print, click "..." andselect the file.

Start

Begins testing.
