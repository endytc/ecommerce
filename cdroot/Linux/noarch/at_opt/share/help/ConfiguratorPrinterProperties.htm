Printer Properties Dialog

Invoke this dialog by clicking Properties... in the Configurator screen or the Printers Configuration screen. This dialog enables you to view and/or edit printer properties.

 

General Tab Name

Unique name for the new printer.
Location

Some location information (optional).
Description

Some additional information about the new printer (optional).
Connection Tab Used port

Select the correspondent PC port used by printer.
Driver Tab Used driver

Select the correspondent printer driver used by printer.
Options

View and/or set specific device options.
Jobs Tab Printer jobs

List of this printer's jobs with correspondent information: ID, title, status, owner's name, creation time, size.
Cancel job

Abort selected job if it's status is Idle or Processing.
Show completed jobs

Shall check if you want to view the list of all printer's jobs including completed an canceled.
Classes Tab Belongs to the

List of printers classes that owns this printer.
Add to Class..

Add this printer to some printers class.
Remove from Class

Remove this printer from selected printers class.
