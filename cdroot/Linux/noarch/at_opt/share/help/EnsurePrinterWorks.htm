How to ensure that printer works

To check if everything OK with your printing system do the following:

  1. Open Configurator.
  2. Switch to Printers configuration.
  3. Select the desired printer in the printers list.
  4. Click the Test... button.
  5. Make sure that the Check printer, the Test device and backend and the Print test page options are checked.
  6. Click the Start button. Testing printer dialog appears.
  7. Check that the Checking driver..., the Checking port... and the Printing test page... options are flagged with "OK". 
  8. Check to see if test page has been printed.

Note: The Test device and backend option is not available for remote printers.

What is going on?

During the printing of test pages the Unified Linux Driver software checks:

  1. If the device you select in printer's list is still available in the CUPS database and is active (you can print to the printer).
  2. If the specified device port is available.
  3. If the device is properly connected and switched on.

In case any step fails further testing cancels and a corresponding error message is shown.
