Unified Linux Driver Configurator

Unified Linux Driver Configurator is a tool primarily intended for configuring printers and MFP devices. Since an MFP device combines printer and scanner in a single case, the Unified Linux Driver Configurator provides options logically grouped around printer and scanner functionality. Besides, there is a special MFP port options' set responsible for regulation of access to MFP printer and scanner via single I/O channel.

With the Unified Linux Driver Configurator you can do the following:

  • View a list of installed printers.
  • Add a printer to your system.
  • Set printers' properties and test printers.
  • View a list of active scanners.
  • Preview and scan images.
  • Process scanned images.
  • Set scanner's options.
  • Control the use of MFP ports.

Each of the Unified Linux Driver Configurator's functional blocks can be accessed by pressing the respective plugin button. Then the right pane of the application displays the appropriate plugin related to printers, scanners or port control features.
