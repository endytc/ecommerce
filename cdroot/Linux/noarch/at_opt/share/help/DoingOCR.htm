Doing Optical Character Recognition

  • In the Tools menu click OCR....

If recognition is successful, the text editor starts and displays recognized text.

By default the external OCR program is gocr, and the external text editor is emacs. To use this command gocr and emacs must be installed.

You can select other applications as the external OCR and the text editor. For details, see Configuring external toolsConfiguring external tools.

Note: Modern Linux OCR programs work relatively well for black English text on white paper, without graphics, scanned in Line Art or Grayscale 150 dpi. Other conditions don't produce acceptable results.
