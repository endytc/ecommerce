Acquiring Image

  1. Insert a page into the scanner.
  2. In the options pane press Scan.

To cancel scanning, press Cancel in the options pane.

The scanned image appears in the new Image Editor window. You may process the image further.

If the image does not fit to available virtual memory, the program asks to create temporary swap file on your home partition. The swap file is intermediate data storage for programs. The size of the temporary swap file is approximately six times larger than the size of the image. The program deletes temporary swap file on exit.

Creating and deleting swap files takes several minutes. If you acquire big images frequently and want to save time, consider extending your computer's memory or increasing the size of Linux regular swap partition.
