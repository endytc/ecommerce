Scanner Properties

Caption

Shows the scanner's model name and port. The caption also has platform-specific controls: system menu, Minimize, Maximize and Close buttons.

Preview Pane

Shows a low-resolution image of the document you are going to scan. Initially the Preview Pane is blank. To generate a preview, press the Preview button. Each successive preview operation replaces the contents of the Preview Pane with a new image. You can drag the mouse within the Preview Pane to select an image area for image acquisition.

Options Bar

The Options Bar controls the scanner attached to the USB or LPT port of your computer. The Options Bar enables you to preview or acquire image. With Options Bar you can customize device-specific options, such as Color Composition, Image Resolution and Scan Area Size and save current option values as Default Options or as a named option set.

Status Line

Shows the estimated width and height of the scan image, and information about selected scanner. Status Line shows a brief explanation of selected menu command when navigating a menu. When the scanning is in progress, Status Line shows the progress indicator.
