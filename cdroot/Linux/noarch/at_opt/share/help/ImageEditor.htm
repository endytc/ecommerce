Image Editor

With Image Editor you can edit the image and deliver it to the final destination: printer, e-mail message, local or remote file.

Caption

Shows the path to the image file. It also has platform-specific controls like the System Menu, the Minimize, the Maximize and the Close buttons.

Main Menu

The Main Menu contains item groups File, Edit, View, Image, Tools. Menu items represented in the toolbar have corresponding icons.

Toolbar

Contains buttons representing frequently used menu commands. Every toolbar button has a tooltip and a status line hint.

Image View

Shows the image you are working with.

Status Line

Shows Brief Image Properties. Status Line shows a brief explanation of selected menu command when navigating a menu. During long operations the Status Line shows the name of operation in progress.

 

When you close Image Manager Main window, all Image Editors are closed as well.
