Storing Scanner Options

You can associate multiple option sets with a device model attached to a given port. The name of the currently selected option set is shown in the combobox at the bottom of the Options Bar.

Restoring default options set

  1. In the Options Bar select Default Job in the Job Type combo box or press Default button.

Default options set represents factory settings for the given device model. You can not modify this set.

Saving named options set

  1. In the Options Bar press Save Option Set As....
  2. Enter the name of the options set in the input box.
  3. Press OK.

Selecting options set

  1. Click the combo box at the bottom of the Options Bar.
  2. Select the name of the option set.

Deleting options set

  1. Select the options set you want to delete.
  2. In the Options Bar press Delete Option Set.

You can not delete the default options set.
