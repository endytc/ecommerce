Monitoring Print Job State

During the print job, you can review the current job's state at any moment.

Current Printing Progress State

A window displays the print job attributes and progress in % during the printing job. To abort the current job click the Cancel button.

Printer's Jobs List

To view all the printer's jobs:

  1. Open Configurator.
  2. Switch to Printers configuration.
  3. Select the desired printer from the printers list.
  4. Click Properties... button. The Printer properties dialog appears.
  5. Switch to the Jobs page.
  6. Click the Cancel job button to cancel the selected job.
  7. Select the Show completed jobs option to see previous (already non-active) jobs in the jobs list.
