Using Automatic Document Feeder (ADF)

  1. Set ADF delimiters of the page width according to the page format selected in the Options Bar.
  2. Insert several pages into ADF.
  3. In the Options Bar press Scan.

To cancel scanning, press Cancel button in the status bar.

Every scanned page will appear in the new Image Editor window.

MFP device with flatbed scanner and ADF

If ADF is empty, the page from flatbed scanner will be scanned once.

If there is at least one page in ADF, only pages from ADF will be scanned.
