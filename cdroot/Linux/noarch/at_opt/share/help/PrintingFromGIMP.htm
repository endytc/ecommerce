Printing from the GIMP

To print image from the GIMP:

  • Ensure that the gimp-print package is installed.
  • In the image window right click the Print command in the File menu.
  • Select the desired printer from the list.
  • Click the Setup printer button. Ensure that the printing command is lpr instead of lp and -orow option is removed.
  • Edit the image options.
  • Click the Print or Print and Save Settings button.
  • In the GUI LPR dialog, edit the printer and print job properties and click Ok to start printing.
