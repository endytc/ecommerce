Main Window

With Image Manager Main Window you obtain the image from scanner or from file. Typically you set up the MFP device for scanning and control the scan process.

 

Main Window contains Caption, Main Menu, and Toolbar in the top part, Status Line, Options Bar, and Preview Pane.

Caption

Displays the name of the application. It also has platform-specific controls: system menu, s Minimize, Maximize and Close buttons.

Main Menu

Contains item groups File, Edit, Tools, View, Help. Menu items represented in the toolbar have corresponding icons.

Toolbar

Contains buttons representing frequently used menu commands. Every toolbar button has a tooltip and a status line hint. The combobox in the left part of the toolbar selects the scanner device and updates Main Window accordingly.

Options Bar

Controls the scanner attached to USB or LPT port of your computer. You typically use Options Bar to start preview or acquisition. With Options Bar you can customize device-specific options, such as Color Composition, Image Resolution and Scan Area Size. You can also save current option values as Default Options or as a named option set.

Preview Pane

Shows a low-resolution image of the document you are going to scan. Initially the Preview Pane is blank. To generate preview, press Preview button. Each successive preview operation replaces contents of the Preview Pane with new image. You can drag mouse within the Preview Pane to select image area for subsequent acquisition.

Status Line

Shows estimated image size in width x height = data size, and information about selected scan device. If the image requires considerable amount of resources - memory, disk space, or time - the image size is shown in red color. Occasionally Status Line shows the hint about selected menu command. When the scanning is in progress, Status Line shows the progress indicator and the Cancel button.

 

The Main Window looks differently depending on scan-capable devices detected at program startup.

If the Image Manager does not find scan-capable devices at startup, the device selection combobox is empty; Options Toolbar and Preview Pane are not shown.

If one or more devices are detected, Main Window shows the device name in the device selection combobox and displays corresponding Options Toolbar and Preview Pane.
