Uploading images to FTP site

  1. On the Tools menu press Upload by FTP.
  2. Enter remote host name (for example, upload.company.com).
  3. Select the port number (usually 21).
  4. Enter remote path (for example, /pub/my_dir).
  5. Enter user name and password.
  6. Press Save As... . Enter the file name and select the image type. They may differ from the name and type of the image you are editing.
  7. If you don't need to keep saved image on disk after uploading, check Delete after upload.
  8. Press Upload.
