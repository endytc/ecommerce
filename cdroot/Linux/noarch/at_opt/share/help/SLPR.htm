GUI LPR Utility

This utility from Unified Linux Driver package replaces standard Linux lpr command and enables you to print documents.

 

Select file(s) to print dialog

This dialog allows you to select one or more files to print if the lpr command is invoked without any command line arguments. Select a document to print and click the Open button.

GUI LPR dialog

It allows you to change options for the current print job.

Printer

Select the destination from list of available printers and classes.

Properties

View and/or edit the selected destination properties. Destination Properties dialog is invoked.

Stop/Start

Toggle the state of printer queue. Stop the printer queue for selected device if it is running or start it otherwise.

Set as User Default

Make the selected destination default for the current user.

Page selection

Select the range of document pages to print. You can choose from All pages, Even pages, Odd pages or Some pages.

Number of copies

Enter the number of copies to print.

Collate copies

Check to print collated copies.

Reverse order

Check to print document pages from last to first.
