Destination Properties Dialog

This dialog is invoked when you click the Properties button in the GUI LPR dialog. It enables you to change options for current print job.

 

General page

Size

Select the page size from list of available for this destination.

Media

Select the media type from list of available for this destination.

Orientation

Select the page orientation - Portrait or Landscape. Check Reverse to reverse image orientaton (rotate to 180 degrees)

Duplex

Select the duplex mode for duplex printing device. See device manual.

Banners

Select the Start and End banners for print job.

Pages per sheet

Select the page count per paper sheet (N-Up printing mode). You can choose 1, 2 or 4 pages per sheet to print.

Text page

This page contains options that apply to plain text documents only.

Margins

Enter the page margins for printed text.

Use device default margins

Check to print text with device default page margins.

Units

Select the measuring units for margin values.

Characters per inch

Enter the characters per inch count (the default value is 10).

Lines per inch

Enter the lines per inch count (the default value is 6).

Columns

Select the text columns count from 1 to 4.

Pretty printing

Check to print text with syntax highlighting and document header.

Graphics page

This page contains options that apply to images only.

Default values

Check to print images with default color options.

Brightness

Enter the brightness value from 0 to 200 (the default value is 100).

Gamma

Enter the gamma value from 0 to 2000 (the default value is 1000).

Hue

Enter the hue value from -360 to 360 (the default value is 0).

Saturation

Enter the saturation value from 0 to 200 (the default value is 100).

Use specific resolution

Check to print images with specific resolution (the default image resolution is 128 pixels per inch).

Use scaling

Check to print images with some scaling. 100% means that image shall be scaled to fit the page.

Fit to page

Set scaling to 100%.

Image position

Set the image position on the page.

Device page

Printing resolution

Select the printing resolution from the list of available resolutions for this destination.

From

Select the input bin.

To

Select the output bin.

Advanced

View and/or set specific device options.
