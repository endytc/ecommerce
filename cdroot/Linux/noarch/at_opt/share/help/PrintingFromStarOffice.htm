Printing from the OpenOffice.org

OpenOffice.org is a collection of office applications. All applications have unified GUI, and use the same printing procedure.

To print form OpenOffice.org application:

  • Click the Print command in the File menu.
  • In the Print dialog select a printer from list, edit printer properties, and click Ok to start printing.
