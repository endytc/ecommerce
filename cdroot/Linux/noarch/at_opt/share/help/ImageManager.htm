Overview

Image Manager is a tool for managing images. Image Manager provides the following capabilities:

  • simple image editing;
  • working with image files on hard disk;
  • image delivery to external media - printing, FTP uploading, sending by e-mail;
  • integration with third-party external tools, like graphics editors and Optical Character Recognition programs.

Image Manager has a Main Window, which have one or more Image Editor tabs.

The Image Editor enables you to process images.. You can update images and manage final destinations (local or remote file, printed copy, e-mail message, or external image-handling tool.)

Closing the Image Manager Main window closes all Image Editors.
