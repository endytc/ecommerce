Toolbars

You can configure toolbars with commands from View menu.

Labels

Shows or hides text labels on the toolbar buttons.

Big Pixmaps

Sets the size of pixmaps (icons) on the toolbar buttons.

Movable Toolbars

Enables or disables toolbars movement.

Save Layout

Saves the positions of movable toolbars. Normally the window saves the layout automatically on closeup. Use Save Layout if you save the layout with the window active.

When the window opens, it has the same layout as last saved window.
