Printers configuration

To view the configuration of the available printers, click button with the printer icon on the left side of the Configurator window.

 

Printers list

Switch to Printers tab in Printers configuration window to see the list of available system printers. Installed printers are displayed as icons with the corresponding printer names. System default printer is marked by string (default). At the bottom of Printers configuration the status bar is displayed. Status bar shows information about currently selected printer: it's status (local/remote, idle/stopped), driver's model and device's URI.

Refresh

Refresh printers and their properties.
Add Printer...

Add new system printer. Invokes Add new printer wizard.
Remove Printer

Remove the selected printer from system.
Set as Default

Make the selected printer system default.
Stop/Start

Stop the selected printer if it is in idle mode or start it in another case.
Test...

Test printer's connection and work. Invokes Testing printer dialog.
Properties...

View or edit the selected printer properties. Invokes Printer properties dialog.

 

Classes list

Switch to Classes tab in Printers configuration window to see the list of available printer classes. At the bottom of Printers configuration window you can see the information about the printer class you have selected.

Refresh

Refresh printers and their properties.
Add Class...

Add new printers class.
Remove Class

Remove the selected printers class from system.
Stop/Start

Stop the selected printers class if it is in idle mode or start it in another case.
Properties...

View or edit the selected printers class properties.
