Passing Images to External Tools

You can extend the functionality of the Image Editor using external third-party programs.

  • External graphics editor allows you, for example, to add text to the image.
  • External Optical Character Recognition (OCR) tool converts the text content of the scanned page to plain text. You can redirect OCR output to the text editor.

You can select external applications and their parameters in the Configure Tools dialog.
