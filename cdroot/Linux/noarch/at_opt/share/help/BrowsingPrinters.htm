Browsing Available Printers

To view and manage available printers:

  1. Open Configurator.
  2. Switch to Printers configuration.
  3. Click the Refresh... button to renew the available printers list.
  4. Click the Add Printer... button to add new printer.
  5. Click the Remove Printer button to remove the printer selected in the printers list.
  6. Click the Set as Default button to make the selected printer your system default.
  7. Click the Start button to start the selected printer or Stop button to stop it.
  8. Click the Test printer... button to ensure that selected printer works.
  9. Click the Properties... button to change properties of the selected printer.
