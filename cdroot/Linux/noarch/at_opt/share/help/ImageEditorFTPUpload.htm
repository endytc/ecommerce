FTP Upload

FTP Upload is intended to upload images to a remote server using FTP protocol. FTP Upload contains the following UI elements:

Remote Server

The Internet domain name or IP address of the remote server, for example upload.company.com.

Port

The FTP port number of the remote server (this number is usually 21)

Remote Path

Path to the directory on the remote server, for example /pub/my_dir.

User name

The login name of the user, also known as account name.

Password

The User's password. The password is shown with asterisks instead of normal characters.

Save As...

Shows the File Dialog. The image file is uploaded under the name you enter in the "Save As" option.

Delete After Upload

When checked, this option deletes the file from disk after successful upload.

Status Line

Shows the information about the uploading process.
