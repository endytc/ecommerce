Unified Linux Driver Package Overview

Unified Linux Driver package is the accompanying software for printers and multifunction peripherals (MFP). The package supports printing and scanning capabilities of MFP on Linux platform.

The Unified Linux Driver driver package is designed to ensure maximum convenience and ease-of-use. The package contains:

  1. Printer and Scanner drivers
  2. Configurator application to manage printers and scan images
  3. Image manager which accepts scanned images and allows simple image edition

The driver package enables you to configure your device and process scanned documents.

The driver package enables you to monitor a number of MFP and/or printer devices via fast ECP parallel ports and USB simultaneously. If you have more than one device, you'll be able to print and scan several documents concurrently.

You can edit and print it on the same local device or send the document as an attachment by email, upload to FTP site or transfer to external OCR system.

The Unified Linux Driver driver package has a smart and flexible installation program. The program detects target Linux clones, sets up proper components and provides appropriate assistance when installing various environments.

If you have problems refer to a Troubleshooting section.
