Printer Properties

To view and change printer properties:

  1. Open Configurator.
  2. Switch to Printers configuration.
  3. Select the desired printer from the printers list.
  4. Click the Properties... button. The Printer properties dialog appears.
  5. Change the name, location or description in the General page.
  6. View or select a port on the Connection page.
  7. View or select a printer driver on the Driver page. Click the Options... button to view or change the default device options.
  8. View the printer jobs list on the Jobs page. Click the Cancel job button to cancel the selected job. Select the Show completed jobs option to see previous (non-active) jobs in the jobs list.
  9. View classes that own this printer on the Classes page. Click Add to Class... button to add this printer to a class. Click the Remove from Class button to remove this printer from the selected class.
  10. Click the Apply button to apply current changes.
  11. Click the Ok button to apply current changes and close the Printer properties dialog.
  12. Click the Cancel button to close the Printer properties dialog without applying changes.

Note:

  • You can change properties of local printers only. Network printers properties are write protected.
  • Some changes like adding to class and removing from class will be applied immediately.
