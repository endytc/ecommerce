Storing and Printing Image Files

Saving the image with new name

  1. On the File menu click Save As.
  2. In the Save File dialog select the destination directory.
  3. Select the file type.
  4. Enter the file title. Save File dialog will automatically append appropriate extension (.bmp, .png and so on) based on file type selected in the File Type combo box.
  5. Press OK.

 

For newly scanned untitled images the command Save also opens Save File dialog.

Saving previously saved image

On the File menu click Save.

Opening the image file

  1. Go to Image Manager Main Window.
  2. On the File menu click Open.

If the image does not fit to available virtual memory, the program asks to create a temporary swap file on your home partition. The swap file is intermediate data storage for programs. The size of the temporary swap file is approximately six times larger than the size of the image. The program deletes temporary swap file on exit.

Creating and deleting swap files takes several minutes. If you open big images frequently and want to save time, consider extending your computer's memory or increasing the size of Linux regular swap partition.

Printing the image file

On the File menu click Print... .
