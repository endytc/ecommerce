Viewing Image Properties

Detailed image properties

  • On Image menu click Properties.

Brief image properties

Move mouse cursor over the image in the Image Editor. Status Line will display the image properties.

Property

Example

Width and Height in pixels

376x519

Zoomed Width and Height in pixels

[752x1038]

Color Bit Depth

8 bits

Current cursor position, as if the image is not zoomed

(180, 241)

Color of the pixel in ARGB format

#ffffffff

Color index in 1-bit or 8-bit palette and number of colors in the palette

255/256 colors

Some properties may be hidden. For example, if the image has 32-bit color depth, it doesn't have the palette, and palette information isn't shown.
