Printing from KDE Editors

KEdit and Advanced Editor are well-known KDE text editors.

 

Printing from KEdit

To print text from KEdit 

  • Click the Print command in the File menu.
  • Select Print directly using lpr.
  • In the GUI LPR dialog select a printer from list, edit printer and print job properties, and click Ok to start printing.

Printing from Advanced Editor

To print text from Advanced Editor:

  • Click the Print command in the File menu.
  • In the Print dialog, set Print system currently used to LPR (Standard BSD print system) and click Ok.
  • In the GUI LPR dialog select a printer from the list, edit printer and print job properties and click Ok to start printing.
