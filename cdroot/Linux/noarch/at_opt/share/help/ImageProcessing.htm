Image Processing

You can process images in the following ways:

  • Use Browse to view image properties and explore the image details using Zoom.
  • Use Save to store information to disk in several formats (BMP, PNG, XBM and others) to open from disk.
  • Use Print to make copies of the images.
  • Use Edit to change the properties of the images.
  • Use Upload to transfer images to the remote site using File Transfer Protocol (FTP).
  • Use Send to send the images by e-mail.
  • Use Pass to external tool for additional processing (OCR or external graphics editor, for example)
