Send Message

Composes an e-mail message and sends it through the default SMTP server. Contains the following UI elements:

From

Your e-mail address. Ex: sender@company.com.

To

Recipient's e-mail address. Ex: recipient@company.com.

Subject

Message subject.

Text

Message text.

Save As...

Shows the File Dialog. Specify name and type of image file to save the image for sending. They may differ from the name and type of the image you are editing.

Delete After Sending

When checked, this option deletes the file from disk after the message is sent successfully.

Send

Sends the e-mail using default SMTP client.
