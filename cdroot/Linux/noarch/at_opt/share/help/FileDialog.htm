File Dialog

Selects a file for opening or saving.

Look in

Contains a path to the file being selected.

One directory up

Opens the upper-level ("parent") directory.

Create New Folder

Creates new directory (folder).

List View

Shows file and directory names with icons.

Detail View

Shows file and directory names plus additional information, such as file size and modification date.

Preview File Contents

Shows preview window that is capable to display the image from the selected file.

File name

Shows the file name. When you are saving the file, be sure that the file doesn't have an extension.

File type

Image type selector. The file extension based on that type will be appended to the file name automatically.

Open/Save

Opens or saves the image file.

Cancel

Cancels the file operation.
