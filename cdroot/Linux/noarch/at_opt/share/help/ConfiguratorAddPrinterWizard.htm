Add Printer Wizard

This wizard is invoked on installation or by clicking the Add Printer... button in the Configurator Printers configuration. It can help you add new printer to the existing printing system.

Add Printer Wizard is the sequence of pages

 

Welcome page

This page shows brief introduction to the wizard. Click Next to continue, Cancel to terminate or Help to show this help window. These three buttons appear on subsequent wizard pages also.

Printers found

Select one of the radio buttons:

  • Local printer
  • Manual select
  • Network printer

If local printer is selected, use combo box with local printer names. Local printers are the printers connected to parallel or USB ports.

Manual select permits enter any valid port name even if no printer or MFP device connected to that port. This option is provided for troubleshooting only.

Search Network Printers

If network printer is selected, press the Search button and then use combo box with network printer names.

Be sure that firewall on your computer is turned off, or network printers search may not work properly. To switch firewall off find firewall initialization script in /etc/init.d/ directory:

grep -l iptables /etc/init.d/*

For example, on Fedora 8, it will be /etc/init.d/iptables script.

Run firewall initialization script without parameters:

/etc/init.d/iptables

All acceptable parameters will be shown. Check firewall status:

/etc/init.d/iptables status

If firewall is running, stop it:

/etc/init.d/iptables stop

Note. If network printers search is performed in Linux OS running in virtual environment, such as VMWare or Xen, the ability of network search is very limited. Please consult your virtual environment manual.

Port for the printer

This page displayed only if Manual select was selected on Printers found page

Select printer port from the list.

For network ports ipp://, socket:// IP address should be specified in the form n.n.n.n where n - decimal integer from 0 to 255

For network port lpd:// value should be specified in the form server/queuename

For network port smb:// value should be specified in the form [[username:password@][workgroup/]]server/name (fields in square brackets are optional)

For file port file: absolute file path should be specified

Driver for the printer

Select printer driver from the list. Preferred driver is highlited

Printer description

Enter printer name, location and short description. Field Name is mandatory. Printer name must be unique. If printer with this name already exists, error message appears.

The last page

The last page of the vizard shows message about installed priter. Click Finish to exit wizard.
